# Calculus I - Limits and Continuity Review

## Lecture snapshot
Professor Lin kept repeating that Calculus starts with Limits because Limits tell us what a function is trying to do near a point, even if the function is undefined exactly there. I wrote down that Continuity is really the promise that the limit, the function value, and the approach from both sides all line up.

## Key ideas
- For a two-sided limit to exist, the left-hand and right-hand Limits have to agree.
- Continuity at x = a means f(a) exists, the limit as x -> a exists, and both values match.
- The epsilon-delta definition is the formal language behind the intuitive graphs we draw in lecture.

## Worked example
For f(x) = (x^2 - 1) / (x - 1), the graph has a hole at x = 1 but the limit still exists. Factoring gives x + 1, so the Limits from both sides go to 2. That means the function is not continuous at x = 1 unless we redefine the missing point.

## Reminder to myself
When I see piecewise questions on the quiz, I should check Continuity by plugging in the boundary value, then compare it to the one-sided Limits. The epsilon-delta proof is still clunky, but I finally see that it is just a precise way to say "I can trap the outputs if I stay close enough on the x-axis."
